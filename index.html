RGB e CMYK

RGB (Red, Green, Blue)

O modelo RGB (Red, Green, Blue), que em português significa Vermelho, Verde, Azul, é um modelo de cores aditivo, ou seja, ele cria cores combinando diferentes intensidades de luz nessas três cores básicas. Esse modelo é muito utilizado em dispositivos eletrônicos como monitores, TVs, câmeras,etc.

Quando essas cores de luz são combinadas em diferentes proporções, elas criam uma variedade infinita de cores. Vermelho, Verde e Azul são chamados de cores primárias de luz no modelo aditivo. Ao misturar essas cores em diferentes intensidades, você consegue criar várias cores. O modelo funciona com intensidades de cada cor, que variam de 0 a 255 (no padrão de 8 bits), ou seja: 0 significa a ausência da cor. 255 significa a intensidade máxima da cor.

  • Vermelho: 255, 0, 0
  • Verde: 0, 255, 0
  • Azul: 0, 0, 255

CMYK (Cyan, Magenta, Yellow, Black)

O modelo CMYK (Cyan, Magenta, Yellow, Key/Black), em português Ciano, Magenta, Amarelo, Preto, é um modelo de cores subtrativo. Esse modelo é usado principalmente em impressão, ou seja, quando você imprime algo em papel ou outros materiais, o modelo CMYK é o mais comum..

No modelo CMYK, as cores são geradas pela subtração das cores primárias de luz, que são Ciano (C), Magenta (M) e Amarelo (Y). O "K" representa o Preto e é adicionado para melhorar a profundidade da cor e garantir que as imagens impressas sejam mais nítidas e com maior contraste. Ciano (C), Magenta (M) e Amarelo (Y) são as cores primárias no modelo subtrativo. Preto (K) é incluído porque, no processo de impressão, as cores primárias não conseguem criar um preto profundo com apenas a mistura delas.

  • Ciano: 100%, 0%, 0%, 0%
  • Magenta: 0%, 100%, 0%, 0%
  • Amarelo: 0%, 0%, 100%, 0%
  • Preto: 0%, 0%, 0%, 100%

Comparação entre os modelos:

    RGB

  • Modelo aditivo, ou seja, as cores são criadas pela adição de luz. Quando as três cores de luz (vermelho, verde e azul) são combinadas em diferentes intensidades, elas geram novas cores. Quando todas as cores estão em intensidade máxima (255 para cada uma), o resultado é branco. Já a ausência de luz (0, 0, 0) resulta em preto.

    CMYK

  • Modelo subtrativo, onde as cores são formadas pela subtração da luz. A ideia é que as cores de tinta subtraem (absorvem) a luz refletida da superfície, resultando em uma cor. A mistura de ciano, magenta e amarelo pode criar várias cores, mas não gera um preto perfeito; por isso, o preto (K) é adicionado para melhorar as sombras e detalhes escuros. Quando as cores estão em intensidade máxima (100% de cada), o resultado é preto (ou um tom muito escuro), enquanto a ausência de tinta resulta em branco.
