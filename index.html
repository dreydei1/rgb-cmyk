Informações sobre RGB e CMYK

  • RGB
  • CMYK

RGB (Red, Green, Blue)

O modelo RGB é utilizado em dispositivos eletrônicos como monitores e câmeras. A combinação das cores primárias (vermelho, verde e azul) gera diferentes cores.

  • Vermelho: 255, 0, 0
  • Verde: 0, 255, 0
  • Azul: 0, 0, 255

CMYK (Cyan, Magenta, Yellow, Black)

O modelo CMYK é utilizado em impressão e representa as cores em termos de ciano, magenta, amarelo e preto.

  • Ciano: 100%, 0%, 0%, 0%
  • Magenta: 0%, 100%, 0%, 0%
  • Amarelo: 0%, 0%, 100%, 0%
  • Preto: 0%, 0%, 0%, 100%
